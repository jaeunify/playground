개인 추천: 일반 사무/게이밍이면 ① Ryzen 5 5600 (29만원)으로 충분합니다. 영상편집·개발·다중작업 등 CPU 성능을 많이 쓰시면 ② Core i5-14400F (41.5만원)가 40% 더 강력합니다.

선택 추가 (권장)

현재 시스템은 SATA SSD 2개 + HDD 1개 구성입니다. B550 보드는 모두 M.2 NVMe 슬롯을 제공하므로, OS를 NVMe SSD로 이전하면 부팅/로딩 속도가 체감상 2~5배 빨라집니다. 기존 BIOSTAR S100(240GB)은 노후화 가능성이 있어 교체를 권장합니다.

추천 부품 상세 금액
Samsung 990 EVO Plus 1TB NVMe Gen4 · 읽기 7,150 MB/s · OS 설치용 135,000
SK hynix Platinum P41 1TB NVMe Gen4 · 읽기 7,000 MB/s · 대안 145,000

※ NVMe SSD 1개만 추가하면 충분합니다. 870 EVO 1TB와 WD Blue 1TB는 그대로 데이터 드라이브로 활용.

참고사항

  • 기존 DDR4 RAM, GTX 1660 SUPER, 저장장치 3종, 파워, 케이스는 모두 재사용 가능합니다.
  • A320 → B550 변경 시 Windows 재설치는 대부분 불필요하나, 최초 부팅 후 칩셋 드라이버 재설치 권장.
  • [AMD 안] B550 보드는 5000 시리즈 지원을 위해 BIOS 업데이트가 필요할 수 있습니다. 구매 시 "Ryzen 5000 Ready" 표기 제품 권장.
  • [Intel 안] B760 DDR4 보드는 반드시 "DDR4" 또는 "D4"가 붙은 모델로 구매해야 합니다. B760/H770에는 DDR5 버전이 따로 있으니 혼동 주의.
  • [현재 보드] BIOSTAR A320MH는 5600 지원이 제한적이고 VRM이 약해 B550 또는 B760 DDR4로 교체 필수.
  • [대체 옵션] 13600KF 선택 시 전력 181W로 기본 쿨러로는 부족합니다. 타워형 공랭 쿨러(3~5만원대) 추가 필요.
  • 5600 / 14400F는 번들 쿨러로 충분히 사용 가능하나, 정숙함을 원하면 사제 쿨러 권장.
  • BIOSTAR S100(240GB)은 TLC/QLC 보급형으로 수명이 짧은 편입니다. CrystalDiskInfo로 건강상태 확인 권장.
  • NVMe SSD로 OS 이전 시 Samsung Data Migration(870 EVO용) 또는 Macrium Reflect Free로 간편 복제 가능.
  • 장기적으로 GPU 업그레이드 계획이 있다면 Intel 안(14400F/13600KF)이 성능 헤드룸 측면에서 유리합니다.
  • 가격은 2026년 4월 기준 시세이며, 실제 구매가와 차이가 있을 수 있습니다.
